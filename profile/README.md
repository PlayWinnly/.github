## About Winnly

Winnly is building a new generation of competitive games where player skill matters.

Our games are designed to be easy to start, difficult to master, and rewarding to improve at. We focus on responsive gameplay, fair competition, strong progression systems, and experiences that keep players coming back.

## What we build

* **Skill-based games** with clear mechanics and room for mastery.
* **Competitive experiences** where players can test themselves against others.
* **Player progression systems** built around improvement, achievements, and rewards.
* **Reliable game infrastructure** that supports fast, fair, and scalable gameplay.
* **Polished cross-platform experiences** with a strong focus on mobile.

## How we work

|                      Player-first                      |                                 Quality-focused                                |                                 Built to scale                                |
| :----------------------------------------------------: | :----------------------------------------------------------------------------: | :---------------------------------------------------------------------------: |
| We start with the player experience and work backward. | We care about responsive gameplay, thoughtful systems, and polished execution. | We build reliable foundations that can support growing games and communities. |

## We are hiring

We are growing the team behind Winnly.

| Role                         | What we are looking for                                                                                                                      |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| **Unity Developers**         | Engineers who care about responsive gameplay, polished mobile experiences, performance, tooling, and maintainable game systems.              |
| **Game Developers**          | Builders with strong instincts for mechanics, game loops, balance, progression, and player experience.                                       |
| **Infrastructure Engineers** | Engineers who can build reliable backend systems, deployment pipelines, observability, multiplayer services, and infrastructure that scales. |
